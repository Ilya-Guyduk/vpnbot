### Какое приложение выбрать

Для VLESS Reality нужен специальный клиент. Рекомендуем по порядку:

- **v2rayNG** — самый популярный, стабильный, активно обновляется
- **Hiddify** — современный интерфейс, проще для новичков
- **NekoBox** — мощный, для продвинутых пользователей

> **Важно:** В Google Play эти приложения периодически скрывают для российских аккаунтов. Если в Play Маркет их нет — устанавливайте APK с GitHub (ссылки ниже). Это безопасно — это официальные релизы от разработчиков.

### Способ 1 — v2rayNG

#### Шаг 1. Установите приложение

Откройте на телефоне страницу релизов:

[github.com/2dust/v2rayNG/releases](https://github.com/2dust/v2rayNG/releases)

Скачайте файл с названием вида `v2rayNG_X.X.X_universal.apk` (последняя версия).

Откройте скачанный APK — Android попросит разрешить установку из неизвестных источников. Разрешите для браузера или файлового менеджера.

#### Шаг 2. Импортируйте ключ

После покупки бот пришлёт вам два варианта:

- QR-код (картинка)
- Текстовая ссылка вида vless://...

**Если у вас QR-код:**

1. Откройте v2rayNG
2. Нажмите «+» в правом верхнем углу
3. Выберите «Импорт конфигурации из QR-кода»
4. Отсканируйте QR с экрана компьютера или другого устройства

**Если у вас текстовая ссылка:**

1. Скопируйте ссылку из бота (долгое нажатие → копировать)
2. Откройте v2rayNG
3. Нажмите «+» → «Импорт конфигурации из буфера обмена»
4. Конфиг добавится автоматически

#### Шаг 3. Подключитесь

1. В списке профилей нажмите на добавленный сервер
2. Нажмите большую круглую кнопку «V» внизу справа
3. Android спросит разрешение на VPN — согласитесь
4. Кнопка станет зелёной — VPN подключён

#### Шаг 4. Проверьте работу

Откройте сайт [2ip.ru](https://2ip.ru) — он должен показать другой IP-адрес и страну.

### Способ 2 — Hiddify

Если интерфейс v2rayNG показался сложным — попробуйте Hiddify. Он более дружелюбен к новичкам.

**Установка:** [github.com/hiddify/hiddify-app/releases](https://github.com/hiddify/hiddify-app/releases) — скачайте файл с пометкой Android.

**Подключение:**

1. Откройте Hiddify
2. Нажмите «Добавить профиль»
3. Выберите «Импорт из буфера обмена» или сканер QR-кода
4. Нажмите «Подключиться»

### Решение проблем

#### VPN подключается, но интернет не работает

Проверьте по очереди:

- Корректность ключа — попробуйте удалить профиль и импортировать заново
- Срок действия подписки в боте — команда /my_key
- Перезагрузите телефон и попробуйте снова
- Попробуйте другой клиент (если был v2rayNG — поставьте Hiddify, и наоборот)

#### Не получается установить APK

Откройте **Настройки → Приложения → Специальный доступ → Установка неизвестных приложений**. Найдите браузер или файловый менеджер, через который пытаетесь установить, и разрешите.

#### Подключается, но некоторые сайты не открываются

Иногда помогает смена DNS внутри клиента. В v2rayNG откройте Настройки → Удалённый DNS → впишите `1.1.1.1` или `8.8.8.8`.

### Полезные мелочи

- В v2rayNG можно настроить автоподключение при старте телефона: Настройки → Опции по умолчанию → «Автоматически подключаться при старте»
- Чтобы VPN работал только для отдельных приложений, включите режим «Per-app proxy» в настройках
- На Android TV v2rayNG тоже работает, но иконку приходится запускать через файловый менеджер
